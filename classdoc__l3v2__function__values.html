doc_l3v2_function_values Class Reference

Detailed Description

  • L3P_PARSE_L3V2_FUNCTIONS_DIRECTLY (value = true): parse the strings rateOf, implies, max, min, quotient, and rem as AST_FUNCTION_RATE_OF, AST_LOGICAL_IMPLIES, AST_FUNCTION_MAX, AST_FUNCTION_MIN, AST_FUNCTION_QUOTIENT, and AST_FUNCTION_REM, respectively.
  • L3P_PARSE_L3V2_FUNCTIONS_AS_GENERIC (value = false): parse the strings rateOf, implies, max, min, quotient, and rem all as AST_FUNCTION with the appropriate name set.
